# ContactHub---Smart-Contact-Manager

This is a web-based Contact Management Application built using HTML, CSS, JavaScript, and Bootstrap. It allows users to efficiently manage their contacts with features such as favorites, emergency contacts, and grouping. The app leverages LocalStorage to persist data across sessions, so all your contacts are saved locally in your browser.

Features

Add, Edit, and Delete Contacts: Easily manage your contact list with a user-friendly modal form.

Favorite Contacts: Mark important contacts as favorites for quick access.

Emergency Contacts: Highlight emergency contacts separately for urgent situations.

Search Functionality: Quickly search contacts by name, phone, or email.

Validation & Alerts: Form inputs are validated with meaningful error messages using SweetAlert2.

Responsive UI: Fully responsive design built with Bootstrap 5, works on desktop and mobile devices.

Dynamic Counters: Real-time counters for total, favorite, and emergency contacts.

LocalStorage Support: All contact data is stored in the browser's LocalStorage for persistence.

Tech Stack

HTML5 & CSS3

JavaScript (ES6)

Bootstrap 5

FontAwesome for icons

SweetAlert2 for notifications
